Commands
$ git - all commands start with git
$ git config - configure the tooling
$ git init - initialize a git local repo
$ git clone - download a project from remote repo
$ git add - prepare a file for staging 
$ git commit - commit a file to the repo (Commits are stored locally until pushed)
$ git add . - prepare all files for staging
$ git log - tracks the author and date of commit
$ git push origin main - use this command to push files to GitHub after commit
$ git fetch - will only look at changes made to the remote repository
$ git pull - will pull all changes to the local repository 

Commands for Branching
$ git branch [branch-name] 
$ git checkout -b [branch-name] - create a new branch and switch
$ git push -u [origin] [branch]
